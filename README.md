雙發 ERP Professional 12.1.2｜POS 鍵盤查詢與數量修改版

POS 操作：
1. 會員欄輸入編號、電話或姓名，例如 1110。
2. 系統即時顯示符合會員，使用 ↑／↓ 選擇，Enter 確認。
3. 商品欄亦可輸入商品編號、條碼、型號或品名，使用 ↑／↓ 與 Enter 選擇。
4. 選好商品後，游標移到數量；輸入數量按 Enter，再修改售價並按 Enter 加入。
5. 購物清單內的數量與單價可直接修改，按 Enter 依序移動。
6. Esc 返回首頁，F8 完成結帳，F11 全螢幕。

請先解壓縮，再用 Chrome 或 Edge 開啟 index.html。
