# 雙發 ERP 11.1 水電材料行專業操作版

本版以傳統水電材料行進銷存軟體的操作邏輯重新優化：固定畫面、鍵盤優先、無須使用滑鼠捲動整個頁面。

## 核心操作
- F1：回主選單
- 方向鍵：選單、欄位、資料列移動
- Enter：進入／確認／開啟資料
- Esc：取消或返回
- F2：新增
- F3：修改
- F5：查詢
- F6／F7：上一筆／下一筆
- F8：儲存
- F9：列印
- F10：首頁
- F11：全螢幕
- PageUp／PageDown：資料表一次上下移動 10 筆
- Home／End：資料表第一筆／最後一筆

## 畫面設計
- 所有功能均為獨立頁面
- 主選單固定在左側且不需捲動
- 工作區整頁固定
- 只有資料表內部可捲動
- 商品、客戶、廠商搜尋結果可直接用鍵盤開啟

請將整個資料夾全部上傳，不能只上傳 index.html。
