# 雙發 ERP 10.1 固定式多頁面版

本版把原本單一畫面改為真正的多頁面架構。左側每一個選項都會開啟獨立 HTML 頁面，頁首、選單與工作區固定。

## 主要頁面

- `index.html` 首頁
- `pos.html` POS 收銀
- `product-search.html` 商品查詢
- `customer-search.html` 客戶查詢
- `supplier-search.html` 廠商查詢
- `products.html` 商品資料
- `customers.html` 客戶資料
- `suppliers.html` 廠商資料
- `purchase.html` 廠商進貨
- `sales.html` 客戶銷貨
- `inventory.html` 庫存管理
- `daily-close.html` 每日結帳
