# 雙發 ERP 10.0.3 Professional

這是已整理成 GitHub 可上傳的專案版本。

## 使用方式

1. 將本資料夾內的所有檔案上傳至 GitHub repository 根目錄。
2. 保留 `index.html` 與 `js` 資料夾的相對位置。
3. 開啟 GitHub Pages 後，以 `index.html` 為首頁。

## 目前資料

- 商品：約 40,762 筆
- 客戶：約 5,304 筆
- 廠商：約 732 筆
- 商品支援多個供應廠商編號與主供應商編號。

## 注意

- 請整個資料夾一起上傳，不要只上傳 `index.html`。
- `js/app-08.js` 內含匯入的商品、客戶與廠商資料。
- 系統仍以瀏覽器本機儲存資料；不同電腦或瀏覽器不會自動同步。
