# 雙發 ERP Professional 12.2

開啟 `index.html` 即可使用，建議使用 Chrome 或 Edge。

## 12.2 主要修正

- POS 會員輸入 `1110` 時，會員編號完全相同者永遠排第一並由 Enter 選取。
- POS 商品、商品資料、客戶、廠商、庫存與進貨查詢統一使用同一套優先規則：
  1. 編號／型號／條碼完全相同
  2. 編號／型號／條碼開頭相同
  3. 名稱等欄位完全相同
  4. 名稱等欄位開頭相同
  5. 其他包含結果
- 方向鍵選擇、Enter 確認。
- POS 數量與單價可直接修改，Enter 後自動重新計算。
- Esc 返回首頁，F8 儲存或結帳，F11 全螢幕。
