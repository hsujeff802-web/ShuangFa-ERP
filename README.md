# 雙發 ERP 11.0 DOS 操作風格版

這一版保留原本商品、客戶、廠商、POS、進銷存與每日結帳資料，重新改成適合門市長時間操作的 DOS 風格固定頁面。

## 操作方式
- F1：回到左側主選單
- 上／下方向鍵：移動選單或資料列
- Enter：進入、選取、確認
- Tab／Shift+Tab：下一欄／上一欄
- Esc：返回或取消
- F2：新增
- F3：修改
- F5：查詢
- F8：儲存
- F9：列印
- F10：回首頁
- F11：全螢幕
